jaks:assemble

Full name:

com.googlecode.jaks:jaks-maven-plugin:0.0.1-SNAPSHOT:assemble

Description:

Assemble a Java application into an archive.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Binds by default to the lifecycle phase: package.

Optional Parameters

Name Type Since Description
appFolder File - The source application folder. Resources in this folder are added to the assembled archives.
Alias: zip-compression
fileSystemTargetFolder File - File-system target. The files from the assembly are copied to this location for use in integration testing and development.
Alias: file-system-target-folder
generatedAppFolder File - Items in the generated app folder are added to the archive.
Alias: generated-app-folder
libFolder String - The folder containing the JARs.
Alias: lib-folder
Default: lib
types List - Types of archives to create. Any of zip, tar, tar.gz, tar.bz2.
Default: ["zip", "tar.gz"]
zipCompression int - For the ZIP archive (if specified in atypes, 0 for no compression, 1 for fastest, 9 for smallest. 2 through 8 are a compromise between fastest and smallest.
Alias: zip-compression
Default: 1
